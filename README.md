# Password-Generator

A lightweight Python script that generates strong, randomized passwords based on custom length and character criteria. I built this project to understand cryptographically sound password policies and data security entry points.

## What it Does
* Generates randomized strings using a mix of uppercase, lowercase, numbers, and special characters.
* Allows the user to specify custom password lengths to meet different security policies.
* Uses Python's internal logic to handle character arrays without relying on third-party dependencies.

## Core Concepts Learnt
* **Character Set Entropy:** Combining multiple character classes to increase structural complexity against brute-force attacks.
* **The `random` Module:** Utilizing random index selection for character array generation.
* **Input Validation:** Ensuring user inputs (like password length) are handled safely without causing runtime crashes.

## How to Run It
Run the script directly through your terminal or command prompt:

```bash
python password_generator.py
```
